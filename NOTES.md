## プロジェクト概要
- **プロジェクト名**：freeder（仮）
- **目的**：Feedlyと連携するRSSリーダー（Windows環境でMacのReeder相当の操作を実現）
- **場所**：/Users/jonji/ClaudeCode/freeder/
- **技術スタック**：Next.js

## 機能要件
- Feedlyと同期（既読・スターの状態を反映）
- 記事一覧のリスト表示
- 既読/未読管理
- スター管理
- キーボードショートカットによる操作

### キーボードショートカット
| キー | 動作 |
|------|------|
| `j` | 次の記事 |
| `k` | 前の記事 |
| `h` | 次のRSSフィード |
| `L` | 前のRSSフィード |
| `b` | 元記事をバックグラウンドのタブで開く |
| `v` | ポップアップでプレビュー |
| `m` | 未読に戻す |
| `s` | スター付け/外し |

### 不要な機能（Mac側で管理）
- カテゴリ管理
- フィード追加/削除

## Feedly連携
- OAuth2認証（Googleアカウント連携）
- 開発時はFeedly開発者トークンを使用し、後からOAuth認証に切り替え
